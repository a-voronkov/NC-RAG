Mock Parser Service

This is a mock parser service for testing Phase 4 of the NC-RAG system.

Available Endpoints:

GET /health

Health check endpoint

POST /jobs

Submit a file for parsing

GET /jobs/{job_id}

Get job status

GET /jobs/{job_id}/result

Get job result (when completed)

Usage:

This mock service simulates the external parser API for development and testing purposes.

All jobs are automatically marked as "queued" → "processing" → "completed" with mock data.
